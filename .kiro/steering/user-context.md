# User Game Preferences

## Project: Just Shapes and Kiro

### Technology Stack
- Vanilla JavaScript (no frameworks)
- HTML5 Canvas for rendering
- Browser-based game

### Game Type
- Auto-scrolling platformer inspired by "Just Shapes and Beats"
- Player character: Kiro logo (kiro-logo.png)
- Obstacle type: Triangle spikes
- Win condition: Survive 60 seconds

### Gameplay Mechanics
- Auto-scroll at constant speed
- Spacebar to jump over obstacles
- Smooth camera following player
- Particle effects on jump and collision

### Visual Style
- Dark theme with purple accents (Kiro brand colors)
- Glowing effects on player and UI
- Parallax background with stars
- Grid-style ground pattern
