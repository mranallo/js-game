# Just Shapes and Kiro

A rhythm-based platformer inspired by "Just Shapes and Beats". Dodge the spikes, survive the level, and reach the end!

## How to Play

1. Open `index.html` in your browser
2. Press **SPACE** to jump
3. Tap for short hops, hold for higher jumps
4. Avoid the red triangle spikes
5. Reach the green flag to win!

## Game Modes

- **Standard Mode** - Complete the level by reaching the end
- **Endless Mode** - Survive as long as possible with increasing difficulty

## Controls

- **SPACE** - Jump (hold for higher jump)
- **Touch** - Jump (mobile support)
- **ESC** - Return to menu (from secret menu)

## Features

### Secret Menu
Click on the word "beat" in the start screen subtitle to access the secret menu. Enter cheat codes to unlock special character skins!

### Skin Selector
Once you've unlocked skins, a skin selector appears on the main menu. Choose between the default Kiro logo or any unlocked skins.

### Available Skins
Unlock these skins by entering codes in the secret menu:
- Lion
- T-Rex
- Gray Cat
- Black Cat
- Orange Cat
- Dog
- Penguin
