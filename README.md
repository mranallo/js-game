# Just Shapes and Kiro

A side-scrolling obstacle dodging game built with vanilla JavaScript and HTML5 Canvas.

## Gameplay

- Dodge spikes by jumping over them
- Survive for 15 seconds to complete the level
- Reach the green flag at the end to win

## Controls

- **Space** - Jump (hold for higher jump, tap for short hop)
- **Touch** - Tap screen to jump (mobile support)

## How to Play

1. Open `index.html` in a web browser
2. Press "Start Game" or hit Space
3. Time your jumps to avoid the red spikes
4. Make it through all obstacles to reach the victory lap
5. Cross the finish line to see the fireworks!

## Features

- Variable jump height (hold Space longer for higher jumps)
- Minimap showing level progress and obstacles
- Parallax scrolling background
- Victory lap with fireworks celebration
- Particle effects on jump and collision
- Mobile touch support

## Project Structure

```
├── index.html    # Game HTML and styling

├── kiro-logo.png # Player sprite

```

## Running Locally

Simply open `index.html` in any modern web browser. No build process or server required.
